# HR Analytics — Gaming Industry

Interactive dashboard for workforce planning & diversity KPIs in the gaming industry.

**Portfolio case study — all data is synthetic. Not affiliated with any company.**

## Features

- Workforce Insights — headcount, turnover, diversity KPIs
- Neurodiversity Impact — inclusion & innovation correlation
- Retention Models — ML churn prediction (Gradient Boosting)
- Performance Dashboard — team & department analytics
- ROI Calculator — cost-of-attrition & programme returns

## Stack

`Python` · `Streamlit` · `Plotly` · `scikit-learn` · `pandas`
